GBA ASM Tutorials


	This tutorial is about coding for the Gameboy Advance using
the Goldroad Assembler. The tutorial is designed to somewhat mirror 
the C coding tutorial at The Pern Project as much
as possible. It is similiar to the above mentioned C tutorial, except that this
is the way to get down and dirty with the actual assembly code.
THIS TUTORIAL DOES ASSUME YOU ARE FAMILIAR WITH GBA CODING IN C
